# MOOTTY
Mootty is a diary application that utilizes text-based sentiment analysis.
Mootty는 일반적인 다이어리가 아닙니다. 당신의 감정을 알고있는 다이어리입니다.

  - 당신의 솔직한 마음을 적으세요.
  - 다이어리 내용에 대한 감정을 알려드립니다.
  - 감정 상태가 어떻게 변하고 있는지 알아볼까요?

### 완료된 기능

  - 이메일 주소를 통한 로그인 기능
  - Layout: Sign in & up
